# Cisco-Packet-Tracer-Project-VIdeo2

## Visual Field Guide (sketchnote)

A hand-drawn, A4-portrait sketchnote infographic explaining Cisco Packet Tracer
end-to-end: what it is, key features, the build workflow, why it matters, an
example small-office lab, and pro tips.

- **SVG (best for sharing / embedding):** [`sketchnote/cisco-packet-tracer-sketchnote.svg`](sketchnote/cisco-packet-tracer-sketchnote.svg)
- **HTML viewer (for crisp hand-drawn fonts + Print-to-PDF):** [`sketchnote/index.html`](sketchnote/index.html)

Open the HTML viewer in any browser and use **Print &rarr; Save as PDF** for a
clean A4 export. The SVG is fully self-contained and renders directly on
GitHub.
